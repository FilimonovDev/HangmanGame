# Виселица на Go

Консольная реализация классической игры "Виселица" на языке Go. Игроку предстоит угадывать случайные слова на русском языке, вводя буквы по одной. Каждая ошибка приближает виселицу к завершению.

## Правила игры

1. Компьютер загадывает случайное слово из списка (файл `words.txt`)
2. Игрок вводит буквы русского алфавита по одной
3. Если буква есть в слове, она открывается на соответствующих позициях
4. Если буквы нет, количество попыток уменьшается, и рисуется следующий элемент виселицы
5. Игра продолжается до тех пор, пока игрок не отгадает слово (победа) или не будет совершено 6 ошибок (поражение)
6. После окончания игры можно начать новую или выйти

## Требования

- Go версии 1.23.5 или выше

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/FilimonovDev/HangmanGame.git
cd HangmanGame 
```

2. Запустите игру:
```bash

go run .
```
